# 主な個人開発

## HomeDash

家庭内LANで動作するローカルダッシュボードアプリです。

家庭内の情報を1画面で管理することを目的としており、  
外部クラウドに依存しないローカル環境での運用を前提に設計しています。

### 主な技術

- Go
- Vue3
- Docker

### Repository

https://github.com/na0chan-go/home-dash

---

## Team Balancer Bot

参加者の情報を元に自動でチーム分けを行う Discord Bot です。

手動で行っていたチーム分け作業を自動化することを目的として開発しました。

### 主な機能

- 参加者登録
- チーム自動生成
- チーム調整
- 運用コマンド（pause / undo / export など）

### 技術

- Go
- Discord API

### Repository

https://github.com/na0chan-go/splatoon-team-balancer-bot
